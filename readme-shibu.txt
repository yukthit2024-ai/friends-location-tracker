This project generates an ANDROID APK whose purpose is to rename files in a SELECTED folder, by suffixing ".xyz" 
The source code was developed in ANTI-GRAVITY using Claude AI
The source code is at repo https://github.com/yukthit2024-ai/file-renamer.git

The file at .github\workflows\android.yml ensure that github action is triggered and android build takes place when pushed to github.
At the end, and APK is generated and available for download at github actions
